# SercueScribe

A FastAPI project with structured layout.

## Project Structure

```
├── app/
│   ├── api/
│   │   ├── endpoints/
│   │   ├── dependencies/
│   ├── core/
│   ├── db/
│   ├── models/
│   ├── schemas/
│   ├── services/
│   ├── utils/
├── tests/
├── requirements.txt
```

## Getting Started

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
